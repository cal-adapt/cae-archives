This file contains multiple versions of the same notebook, used at workshop #1 (_orig). 

The needed data for the visualizations is in this directory, so you can use the _local version without having to access raw data. 

- The _orig version archives what was shown at the workshop, as run on a UCLA server at the time.
- The _readLocal version uses the processed data in this directory.
- The _awsData version accesses the data from AWS -- takes a few mintues, depending on your machine.
- The _awsCluster version tries (and fails) to use a Fargate Cluster to do the computing.

Associated helper functions are in the .py files accompanying these.
