# Cal Adapt Archived Notebooks
This repository contains reference notebooks and scripts for Cal-Adapt, preserved as documentation for users and developers. Contents are not actively maintained but are intended as stable references.

## Status
Notebooks in this repository are not under active development. They are provided as-is for reference. For actively maintained Cal-Adapt tools, see [cae-notebooks](https://github.com/cal-adapt/cae-notebooks).
